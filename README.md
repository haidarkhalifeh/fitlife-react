# FitLife Fitness Application

## Description

FitLife is a fitness web application developed using ReactJS.

The application helps users explore workouts, calculate BMI and calories, and track fitness progress through a dashboard.

## Features

- Home page
- Workout search and filtering
- BMI calculator
- Daily calorie calculator
- Fitness dashboard
- Contact form
- Responsive design

## Technologies Used

- ReactJS
- React Router
- CSS3
- Vite
- Git
- GitHub

## Installation

```bash
npm install
npm run dev
```
